DFS(깊이 우선 탐색)는 그래프에서 한 노드에서 시작해 다음 분기로 
넘어가기 전 해당 분기를 완벽하게 탐색하는 방식이다.

- 자료구조 : 시스템 내부의 **콜 스택(Call Stack)**을 이용한다.(재귀 방식)
별도의 리스트를 append/pop 할 필요 없이, 
함수가 자기 자신을 호출(재귀)하는 것 자체가 스택에 쌓이는 것과 동일하게 동작한다.

- 시간 복잡도: O(V + E) (V: 노드 수, E: 간선 수)

- 필수 코드
```python
import sys
sys.setrecursionlimit(10**6)
```

### 탐색 방법
1. 시작 노드를 스택에 삽입하고 방문 처리한다.
2. 재귀 탐색 수행
Case A : 스택 최상단 노드에 방문하지 않은 인접한 노드가 하나라도 있으면(많을 경우 문제의 요구사항을 읽어보자 ex:작은 노드 부터 방문) 
그 노드를 스택에 넣고 방문처리한다. 
Case B :  방문하지않은 인접 노드가 없으면 스택에서 최상단 노드를 pop한다.
3. 더 이상 2번의 과정을 수행할 수 없을 때까지 반복한다.

### DFS 매서드 정의
1. 그래프/연결 관계 탐색 
- 노드 간의 연결(간선)이 중요하다.
- 문제 키워드: "바이러스가 퍼진다", "연결된 네트워크의 개수", "친구의 친구", "A에서 B로 갈 수 있는가?"
- 핵심 로직: 인접 리스트로 그래프를 만들고, 한 번 방문한 노드는 다시 가지 않는다.

```python
def dfs(graph, v, visited):
    # 1. 현재 노드 방문 처리
    visited[v] = True
    print(v, end=' ')
    
    # 2. 인접 노드 확인
    for i in graph[v]:
        if not visited[i]:
            dfs(graph, i, visited)
```

2. 백트래킹 - 경로/조합 탐색
- 가능한 모든 경우의 수를 전부 훑어봐야 하거나, 특정 조건을 만족하는 조합/순열을 찾을 때
- 문제 키워드 : "합이 K가 되는 숫자 조합 찾기", "모든 경로 다 훑어보기","N개의 원소 중 M개를 고르는 모든 수", "N-Queen"
- 재귀 호출 직후에 방문 처리를 해제하는 것이 핵심이다. 그래야 다른 경로에서 해당 노드를 다시 쓸 수 있다.

```python
def dfs(v):
    visited[v] = True  # 1. 들어갈 때 체크
    # ... 작업 수행 ...
    for next_v in graph[v]:
        if not visited[next_v]:
            dfs(next_v)
    visited[v] = False # 2. 나올 때 체크 해제 (핵심!)
```

3. 2차원 격자(Grid) 탐색
- 지도나 미로 같은 격자판에서 상/하/좌/우로 움직인다.
- 문제 키워드: "N x M 행렬", "상하좌우로 이동", "영역의 넓이" "미로 탈출", "섬의 개수 세기" 등 좌표 기반 문제
- 핵심 로직: dx, dy 방향 벡터를 사용하여 맵의 범위를 벗어나지 않는지 확인하며 재귀를 돈다.
```python
dx = [-1, 1, 0, 0]
dy = [0, 0, -1, 1]

def dfs(x, y):
    if x < 0 or x >= n or y < 0 or y >= m:
        return False
    if graph[x][y] == 0: # 방문하지 않은 노드라면
        graph[x][y] = 1 # 방문 처리
        for i in range(4):
            dfs(x + dx[i], y + dy[i])
        return True
    return False
```
