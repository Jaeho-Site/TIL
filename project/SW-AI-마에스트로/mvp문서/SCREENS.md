## 2. 주요 내비게이션 모델

**하단 탭 바(4개 탭)**가 메인 앱 전반에 유지된다:
`홈` · `탐험` · `히스토리` · `프로필`

- **산책 루프**(추천 → 상세 → 산책 → 완료 → 회고)는 탭이 아니라 **홈에서 실행되는 모달/스택형 플로우**다 — 화면을 점유하고 끝나면 홈으로 돌아온다.
- `설정`은 `프로필`에서 도달한다(탭 아님).
- 스플래시 → (권한 → 로그인) → 홈은 부팅 시 한 번 일어난다.

```
        ┌──────────────────────── 하단 탭 ────────────────────────┐
        │   홈          탐험          히스토리        프로필          │
        └───┬──────────────────────────────────────────────┬──────────┘
            │ (산책 루프 실행, 홈 위에 스택)                  │ 설정
            ▼                                                 ▼
  추천 → 상세 → 산책 → 완료 → 회고 → (홈으로 복귀)
```

## 3. 화면 플로우 (태스크 플로우)

### 3.1 부팅 플로우
```
스플래시 → 위치 권한 → 로그인/회원가입 (또는 건너뛰기=게스트) → 홈
```

### 3.2 핵심 산책 루프 (해피 패스)
```
홈 ──(조건 시트에서 조건 설정)──► 추천
  │                                                │
  │ "다른 코스 보기" ◄─────────────────────────────┤
  ▼                                                ▼
추천 상세 ──(산책 시작)──► 산책 ──(완주/이탈/수동 종료)──►
산책 완료 / 피드백 ──(제출 또는 건너뛰기)──► AI 회고 ──(완료)──► 홈
                                                     │
                                                     └──(코스 저장)──► 프로필 ▸ 저장됨
```

### 3.3 문제 신고 분기 (산책 중)
```
산책 ──("여기 문제")──► 문제 신고 시트 ──(유형 선택)──► 산책으로 복귀
산책 ──(GPS 이탈 자동 감지)──► "길 막힘/없음?" 프롬프트 ──► 산책으로 복귀
```

### 3.4 리텐션 진입점
```
좋은 날 푸시 ──► 홈 (미리 채워진 추천)
히스토리 ▸ 연속 기록/월간 ──► 홈
탐험 ▸ 안 걸은 경로 ──► 추천 상세 (해당 경로)
```

## 4. 화면 명세

> 범례: **CTA** = 행동 유도 버튼. **상태**는 관련 시 로딩 / 빈 상태 / 에러 / 권한 / 오프라인을 다룬다.

---

### 01 · 스플래시
- **목적:** 브랜드 순간; 인증/세션 확인 후 알맞은 진입 화면으로 라우팅.
- **핵심 컴포넌트:** 로고/워드마크, 은은한 자연 모션 또는 정적 브랜드 일러스트, 버전.
- **주 CTA:** — (자동 진행)
- **보조 CTA:** —
- **내비게이션:** → 02 (최초 실행) 또는 → 04 홈 (재방문, 권한 이미 허용됨).
- **상태:** 로딩(기본); 에러(세션/설정 조회 실패 시 재시도).

---

### 02 · 위치 권한 / 온보딩
- **목적:** OS 권한 다이얼로그 **이전에**, 위치가 *지금 여기서* 코스를 추천하는 데 쓰인다고 쉬운 말로 설명(P10, 최소 수집).
- **핵심 컴포넌트:** 설명 그래픽 하나, 1~2문장 가치 설명, "위치 허용" 버튼, 작은 개인정보 안내("산책 중에만 사용; 백그라운드 추적 안 함").
- **주 CTA:** **위치 허용** → OS 다이얼로그 실행.
- **보조 CTA:** **위치 직접 입력**(거부 시 우아한 대체).
- **내비게이션:** → 03 로그인 (또는 로그인이 지연되면 바로 → 04 홈).
- **상태:** 기본; 권한 허용됨; 권한 거부됨(수동 위치 경로 표시, 절대 막다른 길 아님); 영구 거부 시 OS 설정 딥링크.

---

### 03 · 로그인 / 회원가입 (건너뛰기 가능)
- **목적:** 기록/연속 기록/저장 코스가 유지되도록 가벼운 계정 — 그러나 **절대 산책 시작을 막지 않음**(가드레일 #1).
- **핵심 컴포넌트:** 소셜/간편 로그인 옵션, "게스트로 계속하기" 링크, 한 줄 혜택("산책과 연속 기록 유지").
- **주 CTA:** **계속하기**(로그인).
- **보조 CTA:** **게스트로 계속하기**(→ 나중에 프로필에서 로그인 가능).
- **내비게이션:** → 04 홈.
- **상태:** 기본; 로딩(인증); 에러(인증 실패); 게스트 모드(홈에 나중 로그인 유도 배너).

---

### 04 · 홈
- **목적:** 걷기까지의 발사대 — 강력한 기본 추천이 준비되어 있고, 조건에 빠르게 접근(P1, P2).
- **핵심 컴포넌트:**
  - 위치 칩("**[현재 지역]**에서 걷기", 탭해서 변경 가능).
  - **큰 주 카드**: 오늘의 기본/추천 코스 하나 미리보기(썸네일 지도, 이름, 거리, 예상 시간, 목적 태그, 신뢰도 칩).
  - 빠른 조건 칩: 시간 프리셋(10/20/30분), 목적 태그(🌳🤫🦵🛡️) — 탭하면 05 조건 시트 열림.
  - 좋은 날 배너(조건부): "오늘 대전 공기 '좋음' ☀️".
  - 연속 기록 마이크로 위젯(선택, 작게).
  - 하단 탭 바.
- **주 CTA:** **추천 보기** / 큰 카드 탭 → 06 추천 (또는 바로 07 상세).
- **보조 CTA:** **조건 조정** → 05 조건 시트.
- **내비게이션:** → 05, → 06; 탭 → 탐험/히스토리/프로필.
- **상태:** 기본(추천 준비됨); 로딩(위치 파악 / 연산 중); 빈 상태(해당 지역 데이터 없음 → 인근 검증 지역 제안); 권한 거부(수동 위치 유도); 게스트 배너; 오프라인(캐시된 마지막 추천 + 안내).

---

### 05 · 조건 시트 (바텀 시트)
- **목적:** 거의 노력 없이 의도 설정; 모든 것이 합리적 기본값과 함께 선택 사항(P1).
- **핵심 컴포넌트:** 시간 프리셋 + 직접 입력; 거리 슬라이더/프리셋; 난이도(쉬움/보통); **목적 선택기**(🌳🤫🦵🛡️ 다중 선택, 얇은 데이터 목적에는 목적별 신뢰도 힌트); "적용" 버튼.
- **주 CTA:** **코스 보기** → 06 추천.
- **보조 CTA:** **기본값으로 초기화**.
- **내비게이션:** 닫기 → 04 홈; 적용 → 06.
- **상태:** 기본; 조용함/관절 선택 시 얇은 데이터 유의점 표시(P7); 적용됨.

---

### 06 · 추천
- **목적:** 근거 있는 이유와 함께 **하나의** 명확한 추천을 제시(P2, P3) — 빠르게(~3초 체감).
- **핵심 컴포넌트:**
  - 코스 카드: 경로가 그려진 지도 미리보기, 코스 이름, 거리, 예상 시간.
  - **이유 블록**(자연어, 근거 있음): "강변, 조명 밝음, 대체로 평지 — 20분 가벼운 산책에 맞아요."
  - 목적 태그 + **신뢰도 칩**.
  - 주 + 보조 CTA.
- **주 CTA:** **산책 시작** → 08 산책 (또는 07 상세 경유).
- **보조 CTA:** **다른 코스 보기**(재추천; 암묵적 "안 맞음" 기록).
- **내비게이션:** → 07 상세(카드 탭), → 08 산책(시작), ← 홈.
- **상태:** 로딩(~3초 목표 내 스켈레톤); 로드됨; 재추천 중; 대안 없음(폴백 메시지); 얇은 데이터 유의점(P7); 에러(재시도).

---

### 07 · 추천 상세
- **목적:** 결정 전에 사용자가 코스를 이해하게(P3): 핵심 지점, 수치, 주의 사항, 신뢰도.
- **핵심 컴포넌트:**
  - 경로 + **핵심 지점**(출발 / 경유지 / 반환점 / 도착)이 핀 표시된 전체 너비 인터랙티브 지도.
  - 통계 행: 거리, 예상 시간, 난이도, 목적 적합도.
  - **신뢰도 / 한계 표시**(검증됨 vs. 의심 구간; 얇은 데이터 정직성).
  - **주의 사항** 목록(이전 피드백에서 추출된 위험, 있는 경우).
  - 이유(확장).
- **주 CTA:** **산책 시작** → 08 산책.
- **보조 CTA:** **다른 코스 보기** / **저장**(♥ 프로필로).
- **내비게이션:** → 08; ← 06 / 홈.
- **상태:** 기본; 확정 위험 구간 강조; 저장됨; 지도 로딩; 에러.

---

### 08 · 산책 중 (진행)
- **목적:** 최소한의 주의로 핵심 지점을 따라 안내; 빠른 메모 기록; GPS 오차에 대한 통제 유지(P9).
- **핵심 컴포넌트:**
  - 사용자 중심 지도, 경로 + 다음 핵심 지점 강조; 진행 표시(지나온 핵심 지점).
  - 실시간 통계: 경과 시간, 현재까지 거리.
  - **빠른 메모** 어포던스(한 줄 / 음성 선택), 비모달.
  - **"여기 문제"** 버튼 → 09 시트.
  - **수동 "산책 종료"** 항상 표시(가드레일 #6).
- **주 CTA:** **산책 종료**(수동) — 더불어 완주 감지 시 자동 완료.
- **보조 CTA:** **여기 문제**(핀 신고); **일시정지**.
- **내비게이션:** → 09 시트(문제); → 10 완료(완주/수동 종료); 이탈 시 자동 프롬프트("길 막힘/없음?").
- **상태:** 산책 중(기본); 경로 이탈 / 이탈 감지(프롬프트); 약한 GPS(정확도 안내, 수동 종료 의존); 일시정지; 완료 처리 중.

---

### 09 · 문제 신고(핀) 시트
- **목적:** 정확한 GPS 지점에서 저마찰로 정밀한 구간 이슈를 포착(P4, P5) — 최고 품질 데이터.
- **핵심 컴포넌트:** 현재 위치에 자동 드롭된 핀(미니맵에 표시), 문제 유형 태그(`🚧 공사중`, `🚫 길없음`, `🌑 어두움`, `⚠️ 위험`, `🔇 시끄러움`, `+ 직접입력`), 선택적 사진.
- **주 CTA:** **신고**(핀을 가장 가까운 엣지/구간에 매핑; 신뢰도 감소).
- **보조 CTA:** **취소**.
- **내비게이션:** 닫기 → 08 산책으로 복귀.
- **상태:** 기본; 사진 첨부됨; 제출됨(토스트 확인); 위치 불확실(핀 확인 요청).

---

### 10 · 산책 완료 / 피드백
- **목적:** 기억이 가장 생생한 그 순간, 가장 낮은 마찰 단계에서 피드백 포착(P4, P5).
- **핵심 컴포넌트:**
  - 완료 헤더("[코스]에서 좋은 산책이었어요!") + 빠른 통계(거리/시간).
  - **원탭 평가: 👍 / 👎**(필수 최소값).
  - **태그 시트**(특히 👎 시 확장): `🚧🔇🚫🌑⚠️ + 직접입력`; 👍에는 긍정 태그.
  - 선택적 사진 / 자유 텍스트(사다리 최상단, 절대 관문 아님).
- **주 CTA:** **완료**(평가만으로 충분) → 11 회고.
- **보조 CTA:** **상세 추가**(태그/사진/텍스트); **건너뛰기**.
- **내비게이션:** → 11 회고.
- **상태:** 기본(👍/👎); 태그 확장됨; 상세 추가됨; 건너뜀; 제출 중.

### 11 · AI 회고 / 산책 요약
- **목적:** 따뜻하고 진실한 회고로 산책을 보상하고 기록을 확인 — 루프 강화(P6, P8).
- **핵심 컴포넌트:**
  - **AI 회고** 텍스트(근거 있는 요약, 지어낸 사건 없음).
  - 기록된 통계(경로 썸네일, 거리, 시간, 예상 칼로리), 연속 기록 갱신, 획득 배지(있는 경우).
  - 해당 시 **"당신 피드백이 X를 했어요"** 상호성 문구("다음엔 더 활기찬 경로로 고를게요").
  - 코스 저장 및 공유 옵션.
- **주 CTA:** **완료** → 04 홈.
- **보조 CTA:** **코스 저장**(→ 프로필 ▸ 저장됨); **공유**.
- **내비게이션:** → 04 홈; → 12 히스토리; → 14 프로필.
- **상태:** 생성 중(회고 로딩); 준비됨; 배지/연속 기록 축하; 생성 실패(통계 + 재시도 표시, 절대 차단 안 함).

### 12 · 히스토리
- **목적:** 누적된 투자 표시 — 리텐션의 근간(P8).
- **핵심 컴포넌트:** 지난 산책 목록(날짜, 코스, 거리, 평가); **연속 기록** 위젯; **월간 리포트**("7월에 42km — 도심 한 바퀴!"); 필터(목적/지역별).
- **주 CTA:** 산책 탭 → 산책 요약/회고.
- **보조 CTA:** **월간 리포트 보기**.
- **내비게이션:** 탭; → 산책 상세; → 04 홈(다시 걷기).
- **상태:** 기본; 빈 상태(최초: 첫 산책 격려); 로딩; 게스트(히스토리 유지를 위한 로그인 유도).

### 13 · 대전 탐험 (도감 / 지역 지도)

- **주 CTA:** 안 걸은 경로 탭 → 07 추천 상세.
- **보조 CTA:** **배지 보기**.
- **내비게이션:** 탭; → 07 상세.
- **상태:** 기본; 빈 상태(초기: 채울 전체 지도 표시); 로딩; 새 배지 강조.

### 14 · 프로필

### 15 · 설정
- **목적:** 권한, 알림, 개인정보를 투명하게 제어(P10).

- **보조 CTA:** **내 데이터 삭제**; **로그아웃**.

- **상태:** 기본; 권한 변경됨; 푸시 비활성 설명; 파괴적 동작 확인(삭제/로그아웃).
